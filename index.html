Welcome to the Sci-Fi 5e Dungeons and Dragons SRD!

This is a website containing all of the Rules provided by Technomancers of the Coast in the SRD5.1, released under the OGL1.0a.

If you notice any errors, please create a pull request with the correction in the GitHub repo (see footer).

All items in the SRD are on this site. The SRD contains a subset of the content in the official books.

Questions or comments can be directed to xavierck.3d@gmail.com. Theme provided by Reaser.
